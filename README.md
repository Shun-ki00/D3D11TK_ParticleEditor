## 概要
DirectX 11を使用したリアルタイムパーティクルエディターです。
  直感的なGUIでパーティクル効果をリアルタイムに編集・プレビューできるツールです。ゲーム開発や映像制作におけるエフェクト作成を効率化します。

## 動作環境
・OS：Windows 10 / 11  
・GPU : Direct3D 11対応グラフィックカード  
・開発環境：Visual Studio 2022  
・ビルド対象：x64  

## 使用ライブラリ
・Direct3D 11  
・DirectXTK  
・Windows SDK  
・Dear ImGui
